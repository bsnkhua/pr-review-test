# pr-review-test

## Описание проекта

Это тестовый проект для проверки возможности анализировать и тестировать пул-реквесты с использованием специализированных скилов.

## Цель

Проект создан с целью:
- Тестирования функциональности автоматизированной проверки пул-реквестов
- Валидации скилов для code review
- Проверки качества кода при помощи специализированных инструментов

## Структура проекта

- `main` — главная ветка проекта (production)
- `documentation` — ветка с документацией проекта

## Использование

Проект предназначен для тестирования PR review процесса. Все пул-реквесты будут проверяться скилами перед мержем в main ветку.
